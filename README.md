# EmsskyNet

**Tagline:**  
_A sleek, modern web platform that blends e-commerce functionality with a professional portfolio — built for businesses that want to showcase and sell online._

---

## Description

EmsskyNet is a full-stack Django web application developed for **Emmanuel**, combining a mini e-commerce system with a personal portfolio interface. The platform allows users to browse products, explore services, and contact the business directly — all within a responsive and visually clean design.

It’s crafted for performance, scalability, and simplicity, giving small businesses an online presence that looks professional and functions smoothly.

---

## Core Features

- Product and service display with dynamic management
- Django Admin integration for content and inventory control
- Responsive UI built with Bootstrap 5
- Clean product catalog with descriptions and pricing
- Contact section with structured inquiry form
- Optimized static and media handling for deployment
- SEO-friendly metadata and page titles

---

## Tech Stack

- **Backend:** Django 5 (Python)
- **Frontend:** HTML5, CSS3, Bootstrap 5, JavaScript
- **Database:** SQLite (dev) → PostgreSQL (production)
- **Deployment:** Render.com
- **Version Control:** Git & GitHub

---

## Deployment

This project is ready to be deployed on Render.com or similar cloud hosting services. Ensure you configure environment variables (`SECRET_KEY`, `DEBUG=False`, `ALLOWED_HOSTS`, database URL) and run `collectstatic` for static files before deploying.
